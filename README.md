# ConnectSphere 💬

A real-time chat application where users can join with a username and instantly exchange messages.

## 🛠️ Tech Stack
- **Frontend:** React.js, SockJS, STOMP.js — Deployed on Netlify
- **Backend:** Spring Boot, WebSocket — Deployed on Render

## ✨ Features
- Real-time messaging using WebSocket
- Join with custom username
- Clean and minimal chat UI
- Full-stack deployment (Netlify + Render)
